# Task 3 brief

Task 3: Critically assess, select and apply data science tools (MLO3)

The task requires exploratory graphical plots using appropriate Python libraries to uncover insights and communicate them to decision makers.

Dataset: AnAge database animal lifespan records.

Subtask A:
- Read the zipped tab-delimited AnAge dataset as a pandas DataFrame.
- Familiarise with the DataFrame.
- Summarise the number of species/common names within each Animalia class for which maximum longevity information exists.

Subtask B:
- Plot maximum longevity against adult weight.
- Carry this out for the four classes with the most represented species.
- Use logarithmic scales where helpful.
- Discuss whether smaller or larger animals live longer.
- Identify extreme outliers and comment on whether trends are clearer in some animal groups.
- Reflect on how findings may influence ageing research.
